    etspy.base.TomoTilts(data, *args, **kwargs)[source]

    Create a TomoTilts signal instance.

    TomoTilts is a specialized sub-class of Signal1D used to hold information about the tilt values of each projection in a TomoStack.

    Parameters:¶
    data¶

    The signal data. Can be provided as either a HyperSpy hyperspy.api.signals.Signal1D or a Numpy array of the shape (ntilts, 1).

    *args¶

    Additional non-keyword arguments passed to the Signal1D constructor.

    **kwargs¶

    Additional keyword arguments passed to the hyperspy.api.signals.Signal1D constructor.

    Raises:¶

    ValueError – If the data provided does not result in a Signal with signal shape (1,).
